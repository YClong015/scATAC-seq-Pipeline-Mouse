# T-cell dataset — handover note

The T-cell scATAC-seq dataset is the in-house multiome dataset of Mohammadhossein Esmaeili (Nefzger/Patrick/Yu/Chen co-supervision; UQ IMB), generated as part of his PhD project "Shared AP-1-Driven Programs Link T-Cell Exhaustion to Immune Cell Aging."

## Wet-lab generation (do NOT re-run)

Documented in `MEsmaeili-PR2 report.pdf` (Esmaeili PhD progress review 2). Key facts:

| Item | Value |
|---|---|
| Cell type | CD8+ T cells (splenic) |
| Mouse strains | C57BL/6 (young adult, ~2 mo) and aged (~24 mo) |
| Conditions | Young control, Young acute (LCMV Armstrong, cleared), Young chronic (LCMV Clone 13, persistent), Juvenile, Aged |
| Sample prep | Splenocyte isolation → CD8+ enrichment → hashtag-antibody multiplexing → 10x Multiome v1 (RNA + ATAC) |
| Sequencing | Standard 10x Chromium scATAC-seq + GEX libraries |
| Final n cells | ~20,000 high-quality multiome cells (median 10,903 ATAC fragments, 564 GEX genes per cell) |
| HTO column | `deMultliplex2_final_mapped` |

## Bioinformatics handover (computational entry point)

The computational pipeline in this repo starts from a pre-processed Seurat .RData file generated by **Dr Ralph Patrick** (Patrick Lab, IMB UQ):

```
${DATA_ROOT}/Tcells/Tcells_Seurat_filtered.RData
```

Inside the .RData, the Seurat object contains:
- `ATAC` assay (Tn5-fragment counts at sample-specific peaks)
- HTO-demultiplexed `deMultliplex2_final_mapped` metadata column (5 conditions above)
- Per-cell QC: nCount_ATAC, TSS.enrichment, nucleosome_signal, peak_region_fragments

The fragments file required for re-quantification on the universal peak set:
```
${DATA_ROOT}/Tcells/atac_fragments.tsv.gz
```

## Why no raw-data download script

The raw FASTQ files are unpublished and remain under embargo until the Esmaeili manuscript is published. They are not available via GEO/SRA at the time of writing. To use this pipeline on Esmaeili's T cells:

1. Request `Tcells_Seurat_filtered.RData` and `atac_fragments.tsv.gz` from `r.patrick@uq.edu.au` (Patrick Lab, IMB UQ).
2. Place under `${DATA_ROOT}/Tcells/`.
3. Run `01_preprocessing/tcells/Tcell_scATAC.R` to apply re-clustering / Harmony / annotation, then `Tcell_reannotate.R` to apply the corrected cluster→cell-type map and remove B-cell + low-quality contaminant clusters.

For users who have their own CD8+ T-cell scATAC dataset, `Tcell_scATAC.R` accepts any Seurat object with an `ATAC` assay + an HTO/condition metadata column — adjust `RDATA` and `group_col` paths at the top of the script.
